# 🌍 Carbon Footprint Awareness Platform

A web application that helps users calculate, track and reduce their carbon footprint.

## Features

- Carbon footprint calculation
- Environmental impact classification
- Sustainability recommendations
- Responsive user interface
- Secure API integration

## Tech Stack

### Frontend
- React
- Axios
- CSS

### Backend
- Node.js
- Express.js

## Installation

```bash
git clone <repo-url>
cd carbon-footprint-awareness-platform
```

Frontend:

```bash
cd frontend
npm install
npm run dev
```

Backend:

```bash
cd backend
npm install
node server.js
```
