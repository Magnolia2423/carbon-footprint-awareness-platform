# 🌍 Carbon Footprint Awareness Platform

A web application that helps individuals calculate, understand, and reduce their daily carbon footprint. Users can enter information related to transportation, electricity usage, food consumption, and waste generation to estimate their environmental impact and receive sustainability recommendations.

## 📊 Carbon Footprint Calculation

The platform estimates carbon emissions based on:

* Transport Distance (km)
* Electricity Usage (kWh)
* Food Consumption Score
* Waste Generated (kg)

The result is displayed as total estimated carbon emissions in kg CO₂.

## 🌱 Environmental Impact Levels

| Carbon Footprint    | Impact Level       |
| ------------------- | ------------------ |
| Less than 20 kg CO₂ | 🌱 Low Impact      |
| 20 - 49 kg CO₂      | 🌿 Moderate Impact |
| 50+ kg CO₂          | ⚠️ High Impact     |

## 💡 Sustainability Recommendations

The application provides recommendations such as:

* Use public transport or carpooling
* Reduce electricity consumption
* Switch to LED lighting
* Choose sustainable food options
* Recycle household waste
* Use reusable products

## ⚙️ Installation

### Clone Repository

```bash
git clone https://github.com/Magnolia2423/carbon-footprint-awareness-platform.git
```

### Frontend Setup

```bash
cd frontend
npm install
npm run dev
```

### Backend Setup

```bash
cd backend
npm install
node server.js
```
